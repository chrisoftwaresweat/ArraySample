# 🍎 Java Arrays Example (1D, 2D, and User Input)

## 📌 Description
This program demonstrates the basic concept of **arrays in Java**, including:
- 1D arrays
- 2D arrays
- User-defined (sizeable) arrays

An array is a data structure that stores multiple values of the same type in a single variable instead of creating many separate variables.

---

## 🎯 Features
- Creates and prints a 1D String array (fruits)
- Uses both normal for loop and enhanced for loop
- Accesses array elements using indexing
- Demonstrates a 2D array (matrix format)
- Allows user to create a custom-sized array
- Accepts user input and prints the result

---

## 🧠 Concepts Used
- Arrays (1D and 2D)
- For loop
- Enhanced for loop (for-each loop)
- Indexing
- User input (Scanner)
- Dynamic array sizing

---

## ▶️ Sample Flow
1. Program prints fruits using:
   - Normal for loop
   - Enhanced for loop
2. Displays each fruit using index
3. Prints values from a 2D array
4. Asks user for array size
5. Accepts user input values
6. Displays the entered values
