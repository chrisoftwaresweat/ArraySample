# 🍎 Java Arrays Example

## 📌 Description
This program demonstrates the basic concept of **arrays in Java** and how to use different ways to access and print array elements.

An array is a data structure that stores multiple values of the same type in a single variable.

---

## 🎯 Features
- Creates a String array of fruits  
- Prints array using a normal for loop  
- Prints array using an enhanced for loop  
- Accesses and prints elements using indexing  

---

## 🧠 Concepts Used
- Arrays  
- For loop  
- Enhanced for loop (for-each loop)  
- Indexing
